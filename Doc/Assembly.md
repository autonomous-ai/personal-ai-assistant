# Assembly Guide

| Step | Description | Image |
|-|-|-|
| 1 | Assemble the base | ![](../Photo/Assembly/1.jpg) |
| 1.1 | Attach the Cooler to the Raspberry Pi5 board.<br /><br />**Thermal Pads:** Ensure the protective plastic film is removed from the thermal pads on the underside of the cooler before placement. | ![](../Photo/Assembly/2.jpg) |
| 1.2 | **Alignment:** Locate the four mounting holes on the Raspberry Pi 5 board surrounding the main processor.<br /><br />**Securing:** Gently press down on the push pins — ideally in a diagonal pattern — until they click into place to ensure even pressure across the chip. | ![](../Photo/Assembly/3.jpg) |
| 1.3 | Insert Memory card. | ![](../Photo/Assembly/4.jpg) |
| 1.4 | Mount the Raspberry Pi5 board to the Base Plate: locate the four mounting holes on the Raspberry Pi 5 board. | ![](../Photo/Assembly/5.jpg) |
| 1.5 | Align them with the holes on the plate. | ![](../Photo/Assembly/6.jpg) |
| 1.6 | Secure the board to the plate with the M2.3 screws. | ![](../Photo/Assembly/7.jpg) |
| 2 | Prepare the housing. | ![](../Photo/Assembly/8.jpg) |
| 2.1 | - Secure the apex with one M3 x 8mm screw.<br />- Mount the fan to the housing with four M2.6 x 14mm screws. | ![](../Photo/Assembly/9.jpg) |
| 3 | Connect the board and assemble the housing. | ![](../Photo/Assembly/10.jpg) |
| 3.1 | - Plug the fan cable into the 4-pin JST fan connector at the top right of the board. This lets the Pi 5 manage fan speeds automatically based on system temperature.<br />- Connect the PW and GND pins to the momentary push button.<br />- Connect the LED ring and cooling fan as shown. | ![](../Photo/Assembly/11.jpg) |
| 3.2 | Assemble the base plate with the housing. | ![](../Photo/Assembly/12.jpg) |
| 3.3 | Plug in the power supply through the USB-C port. | ![](../Photo/Assembly/13.jpg) |
| 3.4 | Your intern is ready for setup. | ![](../Photo/Assembly/14.jpg) |
